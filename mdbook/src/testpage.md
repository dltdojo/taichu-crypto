## 程式語言 Rust 

Playpen with hello.rs file

{{#playpen rs/time.rs}}

Playpen editable 即時修改功能不像 [Jupyter Notebook](https://jupyter.org/) 多元，目前適合用來學習簡易 Rust 語法。

```rust,editable
fn main() {
    let number = 5;
    print!("{}", number);
}
```

可以探索其他語法的即時編輯功能如 java, golang, nodejs
