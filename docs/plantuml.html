autonumber 301
Alice -> DOH : 可設定可信的 DoH\nHTTPS port 443

隱私與內容過濾

1. DoB 不普及
2. DoB 節點成本需用客戶在乎隱私動機來支付，
   否則容易轉成中心化 API 服務架構，回到紀
   錄竄改100/200原點，還不如用 DoH。
